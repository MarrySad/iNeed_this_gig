Как необходимо использовать этот шаблон. 

Открываем файл index.html
Верстаем там, либо вставляем готовую верстку между КОММЕНТАРИЯМИ
    <!-- ВЕРСТКА ОТ СЮДА -->

    <!-- ВЕРСТКА ДО СЮДА -->

Если необходимо подбавить какую то картинку, добавляем ее в папку MY_IMG
Если необходимо подключить скрипт  кладем в папку MY_JS !!! НЕ ПОДКЛЮЧАЕМ К ШАБЛОНУ СКРИПТЫ HTTPS, СКАЧИВАЕМ И КЛАДЕМ В ПАПКУ.
Если необходимо подключить файл стилей, кладем в папку MY_CSS. 


Свои CSS  Файлы необходимо подключать сверху всех имеющихся  подключенных CSS. желательно так же указать комментарием в  index.html

    <!-- МОИ СТИЛИ -->
    <link href="MY_CSS/style.css" rel="stylesheet">

Точно так же для подключения JS, только JS файл подключаем в самый низ всех скриптов. И тоже желательно сопроводить комментарием.
К подключенным картинкам комментарии не нужны. 


Важно!!! В этот шаблон ничего не брать и нерстать и не изменять, 
его берем в свою  папку и на его основании верстаем.
Текущий шаблон может  изменяться, не значительно, по этому важно перед началом написания новой страницы, взять актуальный шаблон.


Ниже текст для людей в танке. Если ты не в танке, не читай =)

Вынесем важные моменты
1. верстку кладем только в index.html и только между комментариями 
<!-- ВЕРСТКА ОТ СЮДА -->
<!-- ВЕРСТКА ДО СЮДА -->
2. css  подключаем в начало всех имеющихся стилей. и только из папки MY_CSS
3. скрипты подключаем в конец всех скриптов в конце страницы, не подключаем скрипты https, скачиваем, кладем в папку MY_JS
4. Картинки кладем в папку MY_IMG
5. Ничего не трогаем в папках assets css img.
6. Когда будете переходит по пунктам меню. имейте ввиду, они ведут на наш сайт. а не на файлы лежащие рядом с этим файлом. =))
